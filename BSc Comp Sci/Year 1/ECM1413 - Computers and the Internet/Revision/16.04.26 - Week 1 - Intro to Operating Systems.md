

 - An **Operating System (OS)** is a program that controls the execution of programs and acts as an interface between applications and computer hardware.
 - The objectives are to achieve *convenience*, *efficiency*, and the *ability to evolve*.
 - The OS provides services, which are functions that manage hardware, such as file, I/O, process, memory, and multiprocessor management.
 - The OS has a heart, the Kernel, which contains all the hardware management tools mentioned above.

>[!Warning] The OS is a combination of the Kernel plus user-space tools, libraries, and user interfaces (like GUI or shell).
