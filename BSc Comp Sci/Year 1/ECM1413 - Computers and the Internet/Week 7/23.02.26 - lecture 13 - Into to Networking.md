##### What is a Computer Network
 - A group of devices that are connected to provide certain services.
 - A device is some form of computer on the network, e.g. PC, server, mobile phone, etc.
 - **End/Host Devices**:
	 - E.g. laptop, mobile phone, PC, server, printer, etc.
	 - Local vs network printer:
		 - *Local* = directly connected to a single end device.
		 - *Network* = directly connected to the network (switch)
	 - The source and destination of some data.
	 - They have a unique address.
	 - There are two types: 
		 - *Physical*  - Example: MAC (e.g. 3C:22:FB:7A:91:0D).
		 - *Logical*  - Example: IP (e.g. 192.168.1.1).
 - **Network/Intermediary Devices**:
	 - These connect two or more *end devices* together.
	 - They connect hosts to a network.
	 - They include:
		 - **Hub**: Create a network.
		 - **Switch**: Create a network.
		 - **Router**: Joins two networks together.
			 - They contain an ACL (Access Central List)
- A **client** requests a service from a *server*.
- A **server** offers a service for a *client*.
- To access a service, we need a protocol.
	- Web service: HTTP/HTTPs
	- Email: SMTP
	- Remote access: SSH, Telnet
- Peripherals:
	- These are local end devices, i.e. are not directly connected to the network.
	- E.g. local printer, web cam, local scanner.


##### Network Media
 - Wired Connections:
	 - Metallic "Copper" Cable
		 - Uses electrical pulses to transmit data
		 - E.g. ethernet, coaxial
	 - Fibre-optic Cables
		 - Use light to transmit data
		 - E.g. fibre-optic cables
 - Wireless
	 - WiFI


##### Local Area Network (LAN)
 - Covers a relatively limited geographical area.
 - Typically owned or managed by one organisation.
 - You can have a local area network that is made up of multiple small networks connected by a router.


##### Wide Area Network (WAN)
 - Connects two or more LANs together over a large geographical area.
 - A **Internet Service Provider (ISP)** usually connects LANs together to form the WAN.


##### The Internet
 - A global network of interconnected networks.
 - The largest WAN.
 - No one actually owns the internet.
 - A single organisation could own a section of the internet, e.g. BT, but no one can solely own everything.


##### Internet of Things (IoT)
 - This is the idea that everything is connected to the internet.
 - I.e. TVs, smart camera, smart lights, etc, are connected to the internet.


##### Communication Protocols
 - Three layers in human speech:
	 - Content Layer: Forming the data to transmit.
	 - Rules:
		 - Use a common language, i.e. use English to speak.
		 - Wait your turn.
		 - Signal when finished.
	 - Physical Layer: The sound waves coming out of my mouth.
 - A **protocol** is a set of rules that *govern communication*.


##### Protocol Stack (Suite)
 - A set of protocols that work together to enable communication.
 - One example of a protocol stack is:
	 - TCP/IP Stack: This is what is used TODAY!
		 - Application - HTTP, HTTPS, FTP, DNS, SMTP, DHCP, SSH
		 - Transport - TCP, UDP
		 - Internet (IP) - IPv4, IPv6, ICMP
		 - Link (Network Access) - Ethernet, Wi-Fi, PPP, ...
 - A protocol stack is a conceptual model.
 - Another example is OSI:
	 - OSI Reference model:
		 - L7 Application
		 - L6 Presentation
		 - L5 Session
		 - L4 Transport
		 - L3 Network
		 - L2 Data Link
		 - L1 Physical

 - Sender (Encapsulation)
	 - *Application* - Data
		 - You cannot send this in one go, so we use segmentation to dive it up into smaller pieces that are easier to transfer.
		 - Each piece can take a different route.
	 - Transport - 
- **PDU (Protocol Data Unit)** = Data + header
- The process of adding the header to the segmented data is called encapsulation.
