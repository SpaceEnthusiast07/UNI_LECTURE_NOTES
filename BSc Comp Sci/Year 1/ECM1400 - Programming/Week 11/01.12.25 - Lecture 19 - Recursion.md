##### What is Recursion?
 - ***Recursion*** is where a function calls itself to solve a problem
 - Recursion breaks complex problems down into simpler, more manageable subproblems


##### Anatomy of a Recursive Function
 - ***Function Definition*** - The recursive function is defined with parameters to solve the problem
 - ***Base Case*** - A condition that terminates the recursion when met - prevents infinite recursion
 - ***Recursive Case*** - The section where the function calls itself with modified parameters to solve subproblems (divide-and-conquer approach)


##### Example - Factorial
 - Mathematical Definition: $n! = n * (n-1) * (n-2) * \dots * 1$
 - Base Case: $0! = 1$
 - Recursive Case: $n! = n * (n-1)!$
 - Example Python code:
```python
def factorial(n):
	# Base case
	if (n == 0): return 1
	
	# Recursive case
	return n * factorial(n-1)

num = 5
print(f"Factorial of {num} is {factorial(num)}")

--OUTPUT-------------------------------------------------------------
Factorial of 5 is 120
```


##### Understanding the Call Stack
 - Every time a function is called, the the function call is pushed to the stack, and only removed once a base case is met (a return statement)
 - Example Python program to trace recursive calls:
```python
def factorial(n, depth=0)
	# Output the state of the call stack
	print("  "*depth + f"Entering: factorial({n})")
	
	# Base case
	if (n == 0):
		# Output base case message
		print("  "*depth + "Base Case Reached: Returning 1")
		return 1
	
	# Recursive case
	result = n * factorial(n-1, depth-1)
	
	# Unwinding the recursion
	print("  "*depth + f"Returning: {n} * fectorial({n-1}) = {result}")

num = 3
print(f"The factorial of {num} is {factorial(num)}")

--OUTPUT-------------------------------------------------------------
```
