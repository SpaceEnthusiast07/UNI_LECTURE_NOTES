#### Pandas
##### What is Pandas?
 - Built on top of NumPy, designed for *data analysis* and *manipulation*
 - Great for working with *structured/tabular data* (like spreadsheets)
 - Can handle different data types and missing values easily


##### Pandas Data Structures
 1. ***Series*** - 1D labeled array (like one column in Excel)
 2. ***DataFrame*** - 2D labeled table (like a spreadsheet)
 3. ***Flexible***: works with many data types (numbers, strings, etc.)


##### Data Manipulation
 - *Indexing*: access data by row/column labels
	 - `.loc[<rowLabel>,<columnLabel>]` $\rightarrow$ 
 - *Slicing*: extract subsets of rows/columns
 - *Filtering*: use conditions to get specific data


##### Handling Missing Data
 - Find missing (NaN) values
 - Handle them using:
	 - `.fillna()` $\rightarrow$ fill in missing values with 0 (default) for specified value
	 - `.dropna()` $\rightarrow$ remove entire rows that contain at least one missing value
 - Imputation: replace missing values using mean, median or mode


##### Combining Data
 - Merge: combine DataFrames using shared columns
 - Join: like SQL joins (inner, outer, left or right)
 - Concatenate: stack or append DataFrames along rows/columns


##### Importing and Exploring Data
 - Importing - from CSV, Excel, SQL, etc, using `read_csv()` for example
 - Exploring - view structure (`.head()`, `.info()`, `.describe()`)
 - Handle Missing Data: detect and clean up gaps

#### Matplotlib
##### What is Matplotlib
 - Based on the plotting language MATLAB
 - It is a popular Python library used for *data visualisation*
 - It allows you to create a wide variety of plots - from simple *line graphs* to complex *3D charts* - so you can see *patterns*, *trends* and *relationships* in your data
 - It is especially useful alongside *NumPy* and *Pandas*


##### Why Use Matplotlib?
 1. ***Fundamental Building Blocks***
	 - Everything is made of:
		 - *Figure* $\rightarrow$ the whole image or window
		 - *Axes* $\rightarrow$ a single plot area (like one chart inside the figure)
		 - *Plot* $\rightarrow$ the actual data visualised on the axes
	 - This structure give you fine control over how plots look
 2. ***Extensive Customisation***
	 - You can control every detail
		 - Colours, line styles and markers
		 - Axis limits, ticks and grids
		 - Labels, legends and annotations
 3. ***Integration***
	 - Works perfectly with *NumPy* and *Pandas* data


##### Creating Basic Plots
 1. ***Line Plots***
	 - Show data trends over time or sequence
	 - Example: `plt.plot(x,y)`
	 - Useful for time-series or continuous data
 2. ***Scatter Plots***
	 - Show relationships between two variables
	 - Example: `plt.scatter(x,y)`
	 - Great for identifying correlations or clusters
 3. ***Bar Charts***
	 - Compare categories or groups
	 - Example: `plt.bar(categories, values)`
	 - Clear for visual comparisons of discrete items


##### Customising Plot Appearance
 - Colours - choose from named colours, RGB values or built-in palettes
		 - Figure $\rightarrow$ 
 - Annotations - add text, arrows or highlights using `plt.text()` or `plt.annotate()`
 - Grid & Axis Styling - use `plt.grid()`, `plt.xlabel()`, `plt.ylabel()`, `plt.title()` to make plots more readable


##### Example Matplotlib program
```python
# Import NumPy to generate the values
import numpy as np

# Import Matplotlib to visualise the data
# The functions we require live in the PyPlot submodule
import matplotlib.pyplot as plt


# 100 values between 0 and 10
x = np.linspace(0, 10, 100)

# Generate some noisy values for the sine function
# Method:
#    - Generate sine values
#    - For each value, add a random offset between -0.5 and 0.5
y_rand = np.sin(x) + np.random.rand(100) - 0.5

# Generate exact values for the sine function
y = np.sin(x)


# Plot a line graph of the sine function
# Style Format:
#    - "--" means dashed line
#    - "g" means green line
plt.plot(x, y, "--g")

# Plot a scatter graph of the noisy sine values
plt.scatter(x, y_rand, marker="x")

# Change the axes limits
plt.xlim(0,10)
plt.ylim(-1.6,1.6)

# Set the title of the graph
plt.title("Sine Graph")

# Set the axes labels
plt.xlabel("x")
plt.ylabel("y = sin(x)")

# Display a grid in the background
plt.grid(True)


# Save the figure to a SVG file
plt.savefig("figure.svg")
```

##### Multiple Figures and Subplot
 1. *Multiple Figures*
	 - You can open separate windows or figures to show different visualisations
 2. *Subplots*
	 - Use multiple smaller plots within one figure for easy comparison
 3. *Layouts*
	 - Adjust spacing and alignment using `plt.tight_layout()` for neat, professional results


##### Advanced Plotting Techniques
 1. ***Histograms***
	 - Show data distributions
	 - Example: `plt.hist(data, bins=10)`
	 - `bins` 
		 - Integer = number of intervals
		 - List = Width of each interval
 2. ***Heatmaps***
	 - Display numerical data as colours in a grid
	 - Useful for correlation matrices or large datasets
 3. ***Contour Plots***
	 - Show curves of equal values in 2D - often used for mathematical surfaces
 4. ***3D Visualisations***
	 - Plot data in 3D space using: `from mpl_toolkits.mplot3d import Axes3D`
	 - Great for visualising relationships between 3 variables
