##### What are the Heap and Stack?
![[Pasted image 20260310111701.png]]
 - The stack and heap are two ways that Java allocates memory.
 - **Stack** - Stores the *active* methods and their *arguments* + *local variables*.
 - **Heap** - Stores all created *objects*.


##### The Stack in more detail
 - When you *call a method*, a *new frame* is *pushed to* the top of the *Stack*.
 - A **frame** is a *block of memory* that stores the methods *primitive variables* and *object references*.
 - When a *method has completed*, its *frame is popped off* the top of the *Stack*.


##### The Heap in more detail
 - Whenever you *create an object*, all the *data* about that object is *stored* in the *Heap*.
 - *No matter where you create it*, e.g. class variable, member variable, or local variable, it is *still stored in the Heap*.
 - *Objects in the heap* which are *no longer referenced from the stack* are *eligible* for *garbage collection*.
 - **Garbage collection** is a service provided by the Java Virtual Machine (JVM) and is the *process of freeing memory space*.
 - The *garbage collector* in Java *runs on its own thread*, so doesn't impact your programs performance.
 - If you create an Object in a method, its reference is stored in the frame for that particular method on the stack. If you don't return this new Object, it is removed (garbage collector). However, if you return the Object reference, that means the method one frame down in the stack still references it, meaning that Object is not eligible for garbage collection.


##### Example
 - Program:
```java
class Point {
	double x, y;
	
	public Point(double x, double y) {  // Step 3
		this.x = x;  // Step 4
		this.y = y;  // Step 5
	}
	
	public void move(double dx, double dy) {  // Step 11
		this.x += dx;  // Step 12
		this.y += dy;  // Step 13
		String st = "Move done!";  // Step 14
		System.out.println(st);  // Step 15
	}  // Step 16
	
	public void moveLeft(double stepSize) {  // Step 8
		double dx = -1 * stepSize;  // Step 9
		move(dx, 0.0);  // Step 10
	}  // Step 17
}

public class PointApp {
	public static void main(String[] args) {  // Step 1
		Point pt = new Point(3.0, 1.0);  // Step 2
		double stepSize_ = 2.5;  // Step 6
		pt.moveLeft(stepSize_);  // Step 7
	}  // Step 18
}
```
