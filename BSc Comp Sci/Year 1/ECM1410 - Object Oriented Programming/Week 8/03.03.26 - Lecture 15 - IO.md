##### What is an I/O Stream?
 - Inputs and outputs in Java are managed by *streams* of data.
 - A **stream** is a *sequential* and *contiguous* one-way flow of data.
 - The **input stream** is used to *read data* from some source.
 - The **output stream** is used to *write data* to some destination.
 - Example source:
	 - file
	 - keyboard
	 - mouse
	 - network
	 - memory buffer
 - Example destination:
	 - file
	 - monitor
	 - printer
	 - speaker
	 - memory buffer
	 - network


##### Types of Streams used for Files
 - Byte Streams
	 - Each data packet is 8 bits long.
	 - Used to read/write raw bytes directly.
	 - Use a subclass of `InputStream` or `OutputStream`.
 - Character Streams
	 - Read/write data in chunks of 16 bits.
	 - Used for reading and writing to text files.
	 - Use subclass of `Reader` or `Writer`.

##### File Input Stream
 - The `FileInputStream` class is used for reading *byte-oriented data* (streams of raw bytes).
 - Examples: images, audio, video, etc.
 - Using this class:
	 - Create an input byte stream object.
	 - The constructor: 
		 - `public FileInputStream(String name) throws FileNotFoundException`
		 - `public FileNotFoundException(File file) throws FileNotFoundException`
	 - Read one byte of data:
		 - `public int read() throws IOException`
		 - This method reads the the next byte in the stream.
		 - Returns -1 when at the end of the file.
	 - Always close the file once finished with it!
		 - `public void close() throws IOException`
		 - This ensure the file is not corrupted.
 - Example: `FileInputStream out = new FileInputStream("example.txt");`.


##### File Output Stream
 - The `FileOutputStream` class is used for writing *byte-oriented data* (streams of raw bytes).
 - Using this class:
	 - Create an output byte stream object.
	 - The constructor: 
		 - `public FileOutputStream(String name) throws FileNotFoundException`
		 - `public FileOutputStream(File file) throws FileNotFoundException`
	 - Write one byte of data:
		 - `public int write() throws IOException`
		 - This method writes a byte in the stream.
	 - Always close the file once finished with it!
		 - `public void close() throws IOException`
		 - This ensure the file is not corrupted.
 - Example: `FileOutputStream out = new FileOutputStream("example.txt");`.
