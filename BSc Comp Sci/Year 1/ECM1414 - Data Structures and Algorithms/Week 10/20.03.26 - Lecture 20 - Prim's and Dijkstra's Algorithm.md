##### What are Weighted Graphs?
 - When the *edges of a graph have a value* associated with them, this *graph is known as weighted*.
 - These values could represent the time or distance to travel between the two nodes.
 - An example of a weighted graph: ![[Pasted image 20260320190237.png|459]]
 - With weighted graphs, we have two optimisation problems:
	 - Find the lowest-cost tree to connect all nodes (**minimum spanning tree**).
	 - Find the lowest-cost path between any two nodes (**shortest path problem**)


##### What is a Minimum Spanning Tree?
 - The **minimum spanning tree** of an undirected, weighted graph is a *set of edges connecting all nodes* such that the *sum of the weights of the edges is the minimum*.
 - The algorithm used is one of the few algorithms that guarantees the optimum solution even though it uses the greedy approach.
 - Normally, greedy algorithms do not always produce the optimal solution.
 - Idea:
	 - Given any node $x_{0}$, add this node to the partial minimum spanning tree $A = (V_{A}, E_{A})$.
		 - Where $V_{A}$ is the set of vertices in this tree and $E_{A}$ is the set of edges connecting this nodes in the tree.
	 - Find an edge $(x, y)$ with the minimum weight such that $x \in V_{A}$ and $y \not\in V_{A}$.
	 - Add the node $y$ and the edge $(x, y)$ to the partial minimum spanning tree $A$.


##### Prim's Algorithm
 - Prim's algorithm is similar to the breadth-first search algorithm, but instead of a simple queue, a priority queue is used.
 - To implement the queue, we will use a min-heap priority queue.
 - To order this heap, we will use each nodes priority as the value.
 - With priorities, the *lower the value*, the *higher the priority*.
 - And since we want the node with the highest priority at the top, we need the one with the minimum value at the top.
 - *Algorithm*:
	 1. Initialise an empty min-heap priority queue ($Q = \emptyset$).
	 2. Select any starting vertex $u$ for the M.S.T. and insert it into $Q$.
	 3. While $Q \neq \emptyset$ and the vertices in $u$'s adjacency list are not marked as visited, repeat:
		 1. Visit the vertices in $u$'s adjacency list and update $Q$.
		 2. Calculate the minimum among these adjacent vertices.
		 3. Add the minimum to the M.S.T.
		 4. $u$ becomes the minimum vertex just added to the M.S.T.
	 4. Return the M.S.T.
